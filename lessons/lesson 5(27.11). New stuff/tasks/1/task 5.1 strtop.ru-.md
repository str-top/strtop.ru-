### Отчёт по проверке решения

**Описание задачи:**  
create a python script that writes the multiplication table

**Решение студента:**  
```python
print('таблица умножения')
```

**Вердикт:** Неправильно.

**Обратная связь:**  
Код выводит только заголовок, но не таблицу умножения.  
**Действия для исправления:**  
1. Используйте вложенные циклы `for i in range(1, 11): for j in range(1, 11): print(f"{i} * {j} = {i*j}")`.  
2. Форматируйте вывод в таблицу (например, с пробелами или `print(i*j, end=' ')`).  
3. Протестируйте на числах от 1 до 10.
